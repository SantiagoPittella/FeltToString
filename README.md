# FtS README

Convert your Field Element into a String!

## Quickstart

#### Run `cmd + shift + P`.
#### Type: `Felt to String`

Paste a number, it can be in base 10 or 16 with '0x' preffix.

![demo](assets/demo.gif)

You'll see the result in bottom right of the screen:
